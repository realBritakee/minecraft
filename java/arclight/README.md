# Arclight 🚀

## This is a generic egg for the Arclight server software.

Arclight is a specialized Minecraft server implementation that allows you to run both Bukkit/Spigot plugins and modloader mods (Forge, NeoForge, or Fabric) simultaneously. 

The egg supports downloading Arclight for the latest or a specified Minecraft version, as well as downloading a specific Arclight version and choosing your preferred modloader backend.

## Server Ports

The Minecraft server requires a single port for access (default 25565), but plugins or specific mods may require extra ports to be enabled for the server.

| Port       | Default |
|------------|---------|
| Minecraft  | 25565   |
